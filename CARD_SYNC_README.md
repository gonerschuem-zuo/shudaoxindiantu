# 多端同步功能 - 使用说明

## 功能概述

从 v1.05.0 版本开始，书道心电图支持**多端同步**功能。你在任何设备上编辑的卡片信息都会自动同步到云端，确保在电脑、平板、手机上的数据保持一致。

## 启用条件

多端同步功能需要：
1. ✅ **已登录用户账号** - 未登录用户只能使用本地存储
2. ✅ **网络连接** - 需要连接互联网才能同步到云端
3. ✅ **运行数据库脚本** - 管理员需要先执行 `card_sync.sql` 创建数据表

## 如何启用

### 管理员操作（一次性）

在 Supabase SQL 编辑器中运行以下脚本：

```bash
# 在项目目录下
card_sync.sql
```

或在 Supabase Dashboard → SQL Editor 中执行 `card_sync.sql` 的内容。

### 用户操作

1. **登录账号** - 点击右上角"登录"按钮
2. **编辑卡片** - 点击"编辑信息"，修改卡片内容
3. **保存更改** - 点击"保存"，数据会自动同步到云端

## 工作原理

### 数据存储策略

| 用户状态 | 本地存储 | 云端存储 | 同步状态 |
|---------|---------|---------|---------|
| **未登录** | ✅ localStorage | ❌ 不支持 | 仅本地 |
| **已登录** | ✅ localStorage（缓存） | ✅ Supabase | 自动同步 |

### 数据同步流程

```
┌─────────────────┐
│  编辑卡片信息   │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  1. 保存到本地   │ ← 立即生效，离线可用
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  2. 同步到云端   │ ← 需要网络，多端共享
└─────────────────┘
```

### 数据优先级

1. **云端数据** - 最新，多端共享
2. **本地数据** - 缓存，离线可用

登录时，系统会：
1. 优先从云端加载最新数据
2. 使用本地数据作为备份
3. 如果两端数据不一致，以云端为准

## 使用场景

### 场景 1：电脑 → 手机

1. 在电脑上登录，编辑王羲之的卡片信息
2. 点击"保存"
3. 在手机上登录，看到相同的编辑内容 ✅

### 场景 2：手机 → 平板

1. 在手机上登录，修改颜真卿的风格描述
2. 点击"保存"
3. 在平板上登录，看到修改后的内容 ✅

### 场景 3：离线使用

1. 在无网络环境下编辑卡片
2. 数据保存到本地存储
3. 连接网络后登录，数据自动同步到云端 ✅

## 数据冲突处理

如果同一卡片在多个设备上同时编辑：

1. **最后保存者胜出** - 后保存的数据会覆盖先保存的
2. **本地备份保留** - localStorage 中始终保留本地版本
3. **可恢复默认** - 点击"恢复默认"可清除所有自定义编辑

## 隐私与安全

- ✅ **数据隔离** - 每个用户只能看到自己的编辑
- ✅ **RLS 策略** - 数据库行级安全保护
- ✅ **登录验证** - 必须登录才能访问云端数据
- ✅ **删除级联** - 删除账号时自动清除相关数据

## 技术细节

### 数据表结构

```sql
calligraphy_card_settings
├── id (主键)
├── user_id (用户ID)
├── card_id (卡片ID: 1-6)
├── icon (图标 emoji)
├── name (书法家姓名)
├── age (生卒年份)
├── style_title (风格标题)
├── style_desc (风格描述)
└── updated_at (更新时间)
```

### API 端点

- **保存/更新**: `POST /rest/v1/rpc/upsert_card_settings`
- **读取**: `GET /rest/v1/calligraphy_card_settings?user_id=eq.{uid}`
- **删除**: `DELETE /rest/v1/calligraphy_card_settings?user_id=eq.{uid}`

## 常见问题

### Q: 未登录用户能使用吗？
**A:** 可以，但数据仅保存在本地浏览器，不会同步。

### Q: 换设备后数据还在吗？
**A:** 如果已登录，是的！在新设备登录后即可看到之前的编辑。

### Q: 如何清除云端数据？
**A:** 点击"恢复默认"按钮，会同时清除本地和云端的自定义数据。

### Q: 网络断开时能编辑吗？
**A:** 可以，数据会保存在本地。连接网络后再次编辑并保存，数据会同步到云端。
