# 书道心电图

书道心电图，书家的展示舞台

## 项目介绍

一个展示中国历代书法家的网页应用，采用现代化设计风格，结合传统书法美学，提供优雅的浏览体验。

## 功能特色

- **卡片展示**：展示6位中国著名书法家的信息卡片
- **多图上传**：一次可选择多张图片上传，单个卡片最多支持10张图片
- **轮播展示**：自动播放多图轮播，支持手动控制和页码显示
- **信息编辑**：支持编辑所有卡片文字信息
- **图片管理**：删除单张图片、清理缓存等存储管理功能
- **GIF导出**：将多张图片导出为GIF动图

## 技术栈

纯 HTML + CSS + JavaScript（无外部依赖）

## 快速开始

1. 克隆仓库
   ```bash
   git clone https://github.com/gonerschuem-zuo/shudaoxindiantu.git
   ```

2. 打开 `zuoindex.html` 即可使用

## 使用说明

- 点击卡片选中，然后可上传照片
- 支持多图上传，自动启用轮播模式
- 点击"编辑信息"可修改卡片内容
- 鼠标悬停在图片上可删除单张图片
- 多图模式下可导出GIF动图
